## Architecture Decisions

1. **Hybrid pathfinding model**: DistanceField for decisions, waypoints for smooth movement
2. **4-tier map generation**: flow-field → topology → organic serpentine → zigzag fallback
3. **Strategy pattern for movement**: enables future hybrid/distance modes without modifying Enemy
4. **Lateral offsets over fixed lanes**: `pathPoint + offset` prevents lane-swapping jitter
5. **Separate logical/visual position**: pathX/pathZ (movement) vs group.position (rendering)
6. **Event-driven system rebuilds**: InfluenceMap and DistanceField rebuild on tower EventBus events
7. **Setter injection for cross-system deps**: setEffectManager(), setDifficultyScaler() pattern
8. **Road mesh as visual overlay**: grid cells provide gameplay logic, road mesh provides visuals
9. **Instanced rendering + pooling**: 5 draw calls for all enemies, zero GC from spawn/death
